Catalog Skill Card — 5 Propositions

All variants drop the Owners/Consumers rows (still available in the detail modal), keep certifications + tags as quiet footer metadata, and pair the card redesign with a redesigned presentation of the four tiers (Foundational / Core / Advanced / AI & Future). Tier hues are differentiated — stone, blue, violet, teal — instead of today's four near-identical blues. Shown in the approved warm light theme; dark adapts via tokens.

V1Tier Rail — refined current

Evolution, not revolution. Keeps the familiar left-border card but ties it to a stronger section header: tier icon chip, count pill, and a fading color rule. Card keeps icon + name head, description, and gains a dashed-divider footer for certs/tags. Lowest implementation risk.
