My Plan — Category Section Toggles

Each tier becomes an explicit switch-pill: a miniature switch (knob slides, track fills with the tier color), the monochrome tier icon, the name, and a count of skills in the current section. ON = tier-tinted pill, knob right — the whole section (numbered header + its skill cards) is shown. OFF = outlined neutral pill, knob left, muted — the section disappears entirely. This also fixes a logic bug found during review: the page currently re-enables all filters on every render, which silently undoes your toggle — that auto-enable will run only on first load.
