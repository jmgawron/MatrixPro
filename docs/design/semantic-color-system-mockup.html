Semantic Color System — Brand Depth + Achievement Gold

MatrixPro’s shell is navy + accent blue (#5cb1ff / #3b82f6), but functional highlights still pull from unrelated hues — green, cyan, purple, and amber — across 3E icons, category rails, proficiency pickers, matrix cells, and plan status chips. Proposed B (recommended) keeps progression inside the brand family and differentiates 3E with fill weight (ghost → soft → solid gradient), not extra hues. Mastered moves from generic green to achievement gold — distinct from amber warnings.

02 — Design rules

Proposed B — brand-depth hierarchy

Monochrome progression

3E, categories, proficiency, and matrix depth stay on the brand blue ladder. Category rails and proficiency still deepen by weight; 3E icons share one Exposure-style ring — Book, Flask, and Terminal differ by silhouette only.

Outcome semantics

Gold = Mastered / certified complete.
Amber = stale, pipeline, caution only.
Red = danger / destructive only.
Retire green for Mastered — it competed with Education and felt generic.

3E icon presentation

Book · Flask · Terminal silhouettes differentiate Education / Exposure / Experience. All three share the same Exposure-style chrome — soft brand fill + solid ring — so only the glyph changes, not the container.
