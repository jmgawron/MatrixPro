Design Details

  • Background: Vibrant gradient (Cisco blue → purple → pink) creates depth for glassmorphism effect
  • Card Blur: backdrop-filter: blur(12px) creates frosted glass appearance
  • Icon Size: 40px icons in 56px containers (relative to ~280px card = 20% width)
  • Icon Style: Tabler Icons 24×24 grid with 2px stroke, monochromatic white
  • Grid Layout: repeat(auto-fit, minmax(280px, 1fr)) - responsive 3-4 columns
  • Hover Effect: Smooth elevation (translateY -8px) with enhanced shadow
  • Border: Subtle 1px white with 20% opacity for edge definition
  • Shadow Technique: Inset light shadow + outer dark shadow for depth

Icon Library Recommendation

  • Library: Tabler Icons (6,000+ icons, MIT Licensed)
  • Icon Format: Pure SVG, 24×24 grid, 2px stroke width
  • Networking Coverage: router, switch, server, firewall, antenna, cloud, shield, etc.
  • Integration: Download ~15-20 SVGs to frontend/assets/icons/
  • No Build Required: Self-contained, inline SVG embedding, pure vanilla JS
  • Color Inheritance: Icons use stroke="currentColor" for easy theming
