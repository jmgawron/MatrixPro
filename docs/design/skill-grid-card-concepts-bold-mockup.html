Bold evolution of the four card concepts

Each bold variant pushes category color, type weight, and zone separation further — while keeping title, teams, certificate, description, and tags intact. Compare refined (muted) vs bold (amplified) side-by-side on each concept tab.

Bold A — Monument Zones

8px glowing category rail, solid accent icon well, 2-column meta cells, accent-tinted tag footer with 2px top rule.

What makes it bolder

  • Rail width doubles (4→8px) with vertical gradient + glow spill
  • Icon flips to solid accent fill with white glyph (vs soft tint)
  • Meta becomes boxed cells instead of inline labels
  • Tag zone uses full-width accent band — impossible to miss

Trade-off: Tallest bold card (~24px more than refined A). Best when category identity matters more than density.

Bold B — Billboard Hero

Full-saturation hero gradient, 18px/900 title, frosted association strip, tag tray with 5px accent left rail.

What makes it bolder

  • Hero band uses radial + linear gradients at ~2× opacity
  • 58px icon with drop shadow — billboard scale
  • Associations locked in frosted glass strip (visual container)
  • Tag tray gets thick accent spine + inset glow

Trade-off: Highest visual drama — may overpower archived/orphan cards unless states get muted variants.

Bold C — Signal Stack

5px category cap bar, accent-bordered description, split association/tags panels with distinct fill treatments.

What makes it bolder

  • Top gradient cap signals category before you read the title
  • Description gets 3px accent left rule (editorial pull-quote)
  • Associations and tags split into separate bordered panels
  • Full-size chips (not chip-sm) with heavier font weight

Trade-off: More vertical than refined C, but still denser than A/B bold variants.

Bold D — Chromatic Rail

76px full-gradient rail with vertical category label, cert row with icon badge, inverted accent tag strip at bottom.

What makes it bolder

  • Rail becomes a color field (not neutral sidebar) with white-on-accent icon
  • Vertical "CORE" label makes category explicit on every card
  • Cert elevated with dedicated icon badge row
  • Tags bleed to card edge in full-width accent footer

Trade-off: Strongest grid rhythm but rail color competes with section headers — coordinate category palette.
